music-score.js - Documentation

Getting started:

Load music-score.css and music-score.js on your HTML page:

                
    <link rel="stylesheet" href="music-score.css">
    <script defer type="text/javascript" src="music-score.js"></script>

Create an empty div with a unique id:

                
    <div id='sample'></div>

In your script, initialize a MusicScore instance with the div id as argument:

                
    const demoSong = new MusicScore('sample');

API:

.setBars(num_bars)

Add num_bars bars to a MusicScore.

.setTimeSignature(top, bottom)

Set the time signature of a MusicScore to top/bottom

.annotate(bar_num, text)

Add text as an annotation to the bar_numth bar of a MusicScore.

.addNote(bar_num, note, type)

Add a note to the bar_numth bar of a MusicScore. The parameters are described below:

note

The current range of notes supported are 'B3' through 'C6'.

If note is set to 'rest', a rest will be created.

type

The current types of notes supported are 'eighth', 'quarter', 'half', and 'whole'.

.transpose(direction)

Move the notes in a MusicScore. Valid values for direction are 'up' and 'down'

.resetNotes()

Sets the notes in a MusicScore back to their default location, undoing any transposition.

.setTempo(n)

Set the tempo of a MusicScore to n Beats Per Minute.

.tempoUp()

Increments the tempo of a MusicScore.

.tempoDown()

Decrements the tempo of a MusicScore.

.tempoReset()

Sets the tempo of a MusicScore back to the default value.

.getTempo()

Returns the tempo of a MusicScore.

.play()

Begins the Note Player for a MusicScore.
