music-score.js - Examples

Creating a line of music:



HTML:

                
    <div id='sample'></div> // create an empty div with a unique id

JS:

                
    const demoSong = new MusicScore('sample'); // create a new MusicScore object linked to the id of the div
    demoSong.setBars(2); // set how many bars long this piece of music will be
    demoSong.setTimeSignature(5, 4); // set the time signature of this piece of music
    demoSong.addNote(0, 'D4', 'quarter'); // add notes
    demoSong.addNote(0, 'E4', 'quarter');
    demoSong.addNote(0, 'F4', 'quarter');
    demoSong.addNote(0, 'G4', 'quarter');
    demoSong.addNote(0, 'A4', 'quarter');
    demoSong.addNote(1, 'B4', 'quarter');
    demoSong.addNote(1, 'C5', 'quarter');
    demoSong.addNote(1, 'D5', 'quarter');
    demoSong.addNote(1, 'E5', 'quarter');
    demoSong.addNote(1, 'F5', 'quarter');
    demoSong.annotate(0, 'mp');  // add an annotation for dynamic
    demoSong.annotate(1, 'f');

Note player & Changing tempo:



The Note Player highlights the notes in rhythm, allowing the user to play along on their instrument

The user can manually adjust the tempo of the Note Player, in case its too fast or slow for them:

Current tempo: 90 (Beats per minute)

HTML:

                
    <button onclick="playListener()">Click Here to Start the Note Player</button>
    <p>Current tempo: <span id='demo2Tempo'>90</span> (Beats per minute)</p>
    <button onclick="tempoDownListener()">-</button> 
    <button onclick="tempoResetListener()">Reset</button> 
    <button onclick="tempoUpListener()">+</button>

JS:

                
    function playListener() {
        playerSong.play();
    }
    function tempoUpListener() {
        playerSong.tempoUp();
        document.getElementById('demo2Tempo').innerText = playerSong.getTempo();
    }
    function tempoDownListener() {
        playerSong.tempoDown();
        document.getElementById('demo2Tempo').innerText = playerSong.getTempo();
    }
    function tempoResetListener() {
        playerSong.tempoReset();
        document.getElementById('demo2Tempo').innerText = playerSong.getTempo();
    }

Transposing:



The Transposer allows the user to change the key of the music by moving the notes up or down

HTML:

                
    <button class="myButton" onclick="transposeDown()">-</button>
    <button class="myButton" onclick="resetListener()">Reset</button>
    <button class="myButton" onclick="transposeUp()">+</button>

JS:

                
    function transposeUp() {
        transposerSong.transpose('up');
    }
    
    function transposeDown() {
        transposerSong.transpose('down');
    }
    
    function resetListener() {
        transposerSong.resetNotes();
    }
